## Installation

### CocoaPods

SwiftyolLibraries is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SwiftyolLibrary',
:git => 'https://github.com/Murmeko/SwiftyolLibrary.git'
```
